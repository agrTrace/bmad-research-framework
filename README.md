# BMAD Research Framework

🧪 **通用科研智能体框架** - 基于BMAD-METHOD构建的科学研究AI辅助系统

## 特性

- 🤖 **智能配置**: 自动分析项目需求，生成最优专业团队配置
- 📚 **知识驱动**: 基于深度文献调研的结构化知识库
- 👥 **同行评议**: 内置多层级专业评议机制
- 🔄 **动态协作**: 支持多专业智能体并行协作
- 🌐 **通用性**: 适用于任意学科领域的科研项目

## 快速开始

```bash
# 安装框架
npm install -g bmad-research-framework

# 在项目中使用
research-framework install

# 开始研究项目
# 在Claude Code中激活：@research-strategy-planner
```

## 运行方式

- 🌐 **SaaS 在线服务**：`npm run start:saas` 启动 Express API，适合部署到服务器供团队协同使用。
- 🖥️ **Electron 桌面版**：`npm run start:desktop` 启动本地独立界面，无需联网即可浏览目录并生成项目计划。

## 支持的研究类型

- ✅ 国家自然科学基金申请
- ✅ 多学科交叉研究项目
- ✅ 工程技术攻关项目  
- ✅ 社会科学研究项目
- ✅ 基础理论研究项目

## 智能体架构

- **研究策略规划师**: 项目整体规划和可行性分析
- **项目自动配置器**: 智能团队配置生成
- **文献调研专家**: 深度文献调研和知识图谱构建
- **申请报告撰写师**: 标准化学术申请报告撰写
- **同行评议专家**: 多维度专业质量评议
- **专业研究智能体**: 可配置的领域专家模板

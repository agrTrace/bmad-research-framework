# write-nsfc-application

## Task Description

基于文献调研知识库和项目配置信息，撰写高质量的国家自然科学基金申请书核心学术内容。

## Prerequisites

- 已完成文献调研并构建了研究知识库
- 用户已提供项目基本想法和目标
- 已配置项目专业智能体团队
- 可访问相关专业领域知识库

## Task Flow

### Step 1: 知识库信息获取

从研究知识管理器获取必要信息：

```
请提供以下信息以支撑申请书撰写：

1. 文献调研知识库：
   - 研究现状综述
   - 识别的研究空白
   - 关键参考文献
   - 前沿趋势分析

2. 项目基本信息：
   - 项目名称（中英文）
   - 研究目标和内容
   - 拟解决的关键问题
   - 创新点说明

3. 团队基础信息：
   - 前期相关研究基础
   - 已有工作条件
   - 代表性成果
```

### Step 2: 项目摘要撰写

使用 `nsfc-core-application-template.yaml` 的摘要部分：

**中文摘要（400字以内）**：
- 第1句：研究背景和意义
- 第2-3句：现有研究不足和本项目要解决的问题
- 第4-5句：主要研究内容和采用方法
- 第6-7句：预期成果和创新性
- 第8句：学术价值和应用前景

**英文摘要**：
- 对中文摘要的准确翻译
- 保持学术英语的规范性和准确性

**关键词**：
- 中文关键词3-5个，体现研究核心概念
- 英文关键词对应翻译

### Step 3: 立项依据与研究内容撰写

#### 3.1 项目的立项依据

**研究意义**：
- 理论意义：对学科理论发展的贡献
- 实践意义：对实际问题解决的价值
- 社会意义：对社会发展的推动作用

**国内外研究现状**：
基于文献知识库内容：
- 国外研究现状：主要国家和研究机构的进展
- 国内研究现状：国内主要团队的研究水平
- 现有研究的主要成果和技术水平
- 发展趋势和前沿动态

**存在问题**：
基于研究空白分析：
- 理论层面的不足
- 方法技术的局限性
- 实际应用的瓶颈
- 跨学科整合的缺失

#### 3.2 项目的研究内容、目标及关键问题

**研究内容**：
- 研究内容1：具体内容描述和技术路线
- 研究内容2：具体内容描述和技术路线  
- 研究内容3：具体内容描述和技术路线

**研究目标**：
- 总体目标：项目要达成的总体科学目标
- 具体目标：可量化的阶段性目标

**拟解决的关键问题**：
- 关键科学问题1：具体描述和解决思路
- 关键技术问题2：具体描述和解决思路
- 关键方法问题3：具体描述和解决思路

#### 3.3 本项目的特色与创新之处

**理论创新**：
- 在哪些理论方面有突破
- 新理论、新概念、新观点

**方法创新**：
- 新的研究方法或技术手段
- 对现有方法的改进和提升

**应用创新**：
- 新的应用领域或应用模式
- 理论向实践转化的创新途径

### Step 4: 研究方案及可行性分析撰写

#### 4.1 拟采取的研究方案

**总体技术路线**：
- 整体研究框架和逻辑脉络
- 各研究内容间的相互关系
- 从理论到应用的实现路径

**研究方法体系**：
- 理论分析方法
- 实验研究方法
- 数值模拟方法
- 数据分析方法

#### 4.2 技术路线和具体方法

为每个研究内容设计具体技术路线：

**研究内容1的技术路线**：
- 具体实施步骤
- 采用的技术方法
- 预期达到的结果

**关键技术方法**：
- 核心技术的选择理由
- 技术实现的具体方案
- 创新方法的设计思路

#### 4.3 可行性分析

**技术可行性**：
- 所用技术的成熟度分析
- 技术实现的难度评估
- 关键技术的突破可能性

**条件可行性**：
- 现有研究条件的支撑能力
- 所需设备和资源的可获得性
- 团队技术能力的匹配度

**时间可行性**：
- 研究周期的合理性分析
- 各阶段时间安排的科学性
- 里程碑节点的可实现性

#### 4.4 预期困难和解决方案

识别并分析可能遇到的困难：
- 技术难点及应对策略
- 资源约束及解决方案
- 风险因素及控制措施

### Step 5: 研究基础与工作条件撰写

基于用户提供的团队基础信息：

**前期相关研究工作积累**：
- 已有的相关研究成果
- 前期探索性工作基础
- 相关技术积累情况

**已有工作基础**：
- 团队在该领域的工作基础
- 掌握的关键技术和方法
- 建立的研究平台和条件

**代表性成果**：
- 相关领域的重要论文
- 获得的科技奖励
- 承担的相关项目

### Step 6: 预期研究结果撰写

**预期目标和阶段性目标**：
- 项目结束时的总体目标
- 各年度的阶段性目标
- 可量化的考核指标

**预期成果形式**：
- 理论成果：新理论、新方法、新观点
- 技术成果：新技术、新工具、新系统
- 应用成果：示范应用、产业化成果
- 学术成果：论文发表、专著出版

**学术价值和应用前景**：
- 对学科发展的推动作用
- 对相关领域的影响
- 实际应用的可能性和前景
- 经济社会效益预测

**对学科发展的推动作用**：
- 理论体系的完善和发展
- 研究方法的创新和进步
- 应用领域的拓展和深化

## Output Format

生成完整的申请书核心内容，包括：

1. **项目摘要**（中英文 + 关键词）
2. **立项依据与研究内容**
3. **研究方案及可行性分析**  
4. **研究基础与工作条件**
5. **预期研究结果**

## Quality Standards

- [ ] 内容逻辑严密，层次清晰
- [ ] 学术表达规范，语言精练
- [ ] 创新性突出，特色鲜明
- [ ] 可行性分析客观，论证充分
- [ ] 预期成果明确，目标合理
- [ ] 字数控制在规定范围内
- [ ] 格式符合基金委要求

## User Interaction Points

1. **信息确认**：确认获取的知识库信息准确完整
2. **内容审查**：逐部分审查生成内容的准确性
3. **修改建议**：根据用户反馈调整和优化内容
4. **最终确认**：确认最终版本满足要求

## Success Criteria

- 申请书内容完整，符合基金委格式要求
- 立项依据充分，基于扎实的文献调研
- 研究方案清晰可行，技术路线合理
- 创新性表述准确，特色突出
- 预期成果明确，与研究内容匹配
